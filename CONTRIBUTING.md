# Guia de Contribuição

Obrigado por contribuir com a Plataforma Inteligente de Diagnóstico por Imagem (PIDI).

## Fluxo de Desenvolvimento

1. Criar uma branch para cada funcionalidade.
2. Desenvolver.
3. Executar testes.
4. Criar Pull Request.
5. Revisão.
6. Merge.

## Padrões

- TypeScript
- Clean Code
- SOLID
- DDD
- Clean Architecture

## Commits

Utilizar Conventional Commits.

Exemplos:

feat:
fix:
docs:
