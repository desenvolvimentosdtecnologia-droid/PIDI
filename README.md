# PIDI - Plataforma Inteligente de Diagnóstico por Imagem

> Plataforma moderna de Diagnóstico por Imagem desenvolvida para a **UDI - Uniclinic Diagnóstico por Imagem**, com integração ao sistema Naja, PACS, DICOM Modality Worklist, Inteligência Artificial e Portais Web.

---

# Objetivo

A PIDI tem como objetivo substituir gradualmente os módulos operacionais do RIS, mantendo integração com o sistema Naja e oferecendo uma plataforma moderna, modular e escalável para toda a operação da clínica.

---

# Principais Funcionalidades

- Cadastro de Pacientes
- Cadastro de Exames
- Integração com Naja
- DICOM Modality Worklist
- Recepção de Imagens DICOM
- Integração PACS
- Editor Inteligente de Laudos
- Assinatura Digital
- Portal Médico
- Portal do Paciente
- Dashboard Gerencial
- Inteligência Artificial
- Auditoria
- Controle de SLA
- Workflow Clínico
- Notificações
